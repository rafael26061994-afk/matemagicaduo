Matemágica (V2) — Trilhas + Prática + Painel do Professor
==================================================

ARQUIVOS
- index.html
- style.css
- script.js

COMO USAR (ALUNO)
1) Abra o arquivo index.html no navegador (Chrome/Edge).
2) Clique em “Perfis” → crie um perfil (nome, ano, turma, escola).
3) Vá em “Trilhas” ou “Prática”.
4) Para o professor acompanhar: Home → “Exportar progresso”.
   Isso baixa um arquivo .json com o progresso do aluno.

EVENTO SEMANAL
- “Semanal” tem aquecimento (3 questões) e chefão semanal (8 questões).
- O tema alterna automaticamente por semana: padrões / paridade / contagem.
- São itens no estilo OBMEP (não são questões oficiais).

COMO USAR (PROFESSOR)
1) Abra o index.html no seu computador.
2) Vá em “Professor”.
3) Importe os arquivos .json exportados pelos alunos (pode selecionar vários).
4) Selecione Escola e Turma para ver a tabela.
5) Use:
   - “Exportar CSV” (planilha com indicadores por aluno)
   - “Gerar relatório (texto)” (resumo da turma, pronto para copiar/baixar)

OBSERVAÇÕES
- Tudo funciona offline. Os dados ficam no navegador (localStorage).
- Se abrir em outro computador, será um “novo” localStorage: por isso o fluxo de exportar/importar.
